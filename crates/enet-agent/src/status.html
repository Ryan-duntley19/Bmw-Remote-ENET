Desktop connection

The Client auto-detects the desktop by pair code — no static IP needed. If Desktop stays Waiting, click Auto-find, or enter a LAN IP from http://127.0.0.1:47901/ as a one-time hint.

ISTA on the desktop: needs Npcap on both PCs, BMW-ENET adapter (169.254.1.1) on the desktop, and ENET cable + ignition on the laptop. Tunnel “Connected” alone is not enough for ISTA.
